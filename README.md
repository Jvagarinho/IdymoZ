# IdymoZ 🤖🌍

> Seu tutor de inglês com inteligência artificial

IdymoZ é uma aplicação web inovadora desenvolvida para ajudar falantes portuguêsesexercitar e melhorar o seu inglês através de conversas inteligentes com IA. Adaptável a diferentes níveis de proficiência, oferece uma experiência de aprendizado personalizada e interativa.

---

## ✨ Funcionalidades

### 🎯 4 Níveis de Proficiência

| Nível | Descrição |
|-------|-----------|
| **Explorer** (A1/A2) | Vocabulário simples, frases curtas, traduções em português quando necessário |
| **Adventurer** (B1) | Conversa diária, phrasal verbs comuns, correções suaves |
| **Master** (B2) | Temas complexos, idioms, vocabulário profissional, feedback estruturado |
| **Expert** (C1/C2) | Debates filosóficos/técnicos, linguagem académica avançada |

### 🗣️ Reconhecimento de Voz
- Use o microfone para falar diretamente em inglês
- Pratique sem precisar digitar

### 🔊 Síntese de Voz
- Ouça as respostas da IA em inglês
- Velocidade e tom ajustados automaticamente conforme o nível

### 🎨 Design Adaptativo
- Temas visuais que mudam conforme o nível selecionado
- Interface moderna, elegante e responsiva

## 🚀 Como Usar

### 1. Obtenha uma API Key
 Você precisará de uma chave API da Groq:
- Aceda a [groq.com](https://groq.com)
- Crie uma conta gratuita
- Gere a sua API Key

### 2. Execute a Aplicação
Clone o repositório e abra o ficheiro `index.html` no seu navegador favorito:

```bash
git clone https://github.com/Jvagarinho/IdymoZ.git
cd IdymoZ
# Abra index.html no seu navegador
```

### 3. Configure a API Key
Na primeira utilização, insira a sua API Key quando solicitado. A chave será guardada automaticamente no seu navegador.

### 4. Comece a Praticar!
- Selecione o seu nível de proficiência
- Clique no microfone para falar ou digite a sua mensagem
- Receba feedback instantâneo da IA
- Ouça as respostas para melhorar a sua compreensão oral
